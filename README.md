## Prerequisites

- Node v16+
- npm v8+

## Installation

`npm install`

## Running the app

`npm run dev`

## API Access

`cp .env.sample .env`

- Add your API key to `VITE_API_KEY`
- Add your API access token to `VITE_API_ACCESS_TOKEN`
